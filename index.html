HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer section tipically contains information about the author, copyright data or links related to documents.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is a design language that makes a website look more appealing than just plain or uninspiring pieces of text
  • CSS soecifies color, alignment, spacing, decoration, transformation.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system used for tracking changes in computer files.
  • You do not need GitHub to use git, but you cannot use GitHub without using git.

JavaScript

  • Javascript is a text based programing language.
  • Each line of JavaScript code ends with a semicolon.
  • The order of code affects the outcome.
  • Control flow is the order in which a computer executes code in a file.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
